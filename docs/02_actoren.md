# 2. Actoren van het systeem

Actoren zijn alle partijen die interactie hebben met de applicatie.
| Actor | Rol |
|------|-----|
| Klant | Gebruikt de app |
| B2B klant | Plaatst bestellingen |
| Klantenservice medewerker | Ondersteunt klanten |
| Admin | Beheert het systeem |
| AI chatbot | Geeft automatische antwoorden |
| ERP systeem | Verwerkt bestellingen en voorraad |
| CRM systeem | Beheert klantdata |
| BI tool | Analyseert data |

In de casus werken deze actoren samen binnen één systeem. De klant staat centraal en gebruikt de app, terwijl systemen zoals ERP en CRM op de achtergrond data leveren. Dit zorgt voor een geïntegreerde oplossing waarin alle processen samenkomen.
