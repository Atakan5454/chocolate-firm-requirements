# 5. User stories

## Product scannen
Als klant wil ik een product scannen zodat ik productinformatie kan zien  

Acceptatiecriteria:
- QR-code wordt herkend  
- Productinformatie wordt weergegeven  

Inschatting: M  

---

## Dashboard
Als klant wil ik mijn producten zien zodat ik overzicht heb  

Acceptatiecriteria:
- Producten zichtbaar  
- Houdbaarheid zichtbaar  

Inschatting: M  

---

## Klacht indienen
Als klant wil ik een klacht melden zodat mijn probleem opgelost wordt  

Acceptatiecriteria:
- Formulier beschikbaar  
- Upload van foto mogelijk  
- Status zichtbaar  

Inschatting: L  

---

## Chatbot
Als klant wil ik vragen stellen zodat ik snel antwoord krijg  

Acceptatiecriteria:
- 24/7 beschikbaar  
- Antwoorden correct  

Inschatting: L  

---

## Bestellen
Als klant wil ik producten bestellen zodat ik direct kan kopen  

Acceptatiecriteria:
- Product selecteren  
- Betaling werkt  
- Orderbevestiging  

Inschatting: XL  

---

## Aanbevelingen
Als klant wil ik aanbevelingen krijgen zodat ik nieuwe producten ontdek  

Acceptatiecriteria:
- Gepersonaliseerde aanbevelingen  

Inschatting: L
