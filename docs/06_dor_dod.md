# 6. Definition of Ready en Definition of Done

## Definition of Ready
- User story is duidelijk beschreven  
- Acceptatiecriteria aanwezig  
- Inschatting mogelijk  
- Klein genoeg voor een sprint  

## Definition of Done
- Functionaliteit werkt  
- Getest en goedgekeurd  
- Acceptatiecriteria gehaald  
- Geen bugs aanwezig
